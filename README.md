# crystal-chef

A client library for interacting with Chef servers (super alpha work in progress)

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  crystal-chef:
    github: sylvainsf/crystal-chef
```

## Usage

```crystal
require "crystal-chef"
```
